### Archivo: Nginx config

```
# /etc/nginx/sites-available/analyxa
server {
    listen 80;
    server_name analyxa.ai www.analyxa.ai;

    root /var/www/analyxa;
    index index.html;

    location / {
        try_files $uri $uri/ =404;
    }

    # Cache static assets
    location ~* \.(css|js|png|jpg|jpeg|gif|ico|svg|woff2?)$ {
        expires 30d;
        add_header Cache-Control "public, immutable";
    }

    # Security headers
    add_header X-Frame-Options "SAMEORIGIN" always;
    add_header X-Content-Type-Options "nosniff" always;
}
```

**Setup nginx:**
```bash
# Crear config
sudo cp /path/to/analyxa-nginx.conf /etc/nginx/sites-available/analyxa
sudo ln -sf /etc/nginx/sites-available/analyxa /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx

# Si el dominio está configurado, agregar HTTPS con certbot:
```
